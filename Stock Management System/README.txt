Needs to Install in computer

1. Install MySql.Data nuget
- Open the file in Visual Studio
- Locate the Tools tab on upper part of the Visual Studio
- Click the NuGet Package Manager
- Click Manage NuGet Packages for Solution
- Click Browse and Search for MySql.Data by Oracle and then Install

2. Install Nuget.Frameworks by Microsoft
- While on Manage NuGet Packages for Solution
- Search for NuGet.Frameworks by Microsoft and then Install

3. XAMPP
- Install XAMPP in your computer
- Start the Apache and MySQL
- Click the Admin button in MySQL
- Click the Import on the upper part of phpMyAdmin
- On the File Import, Click the Choose File and then click the stockmanagement sql file inside the Database Folder
